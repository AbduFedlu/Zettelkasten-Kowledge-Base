# Obsidian Knowledge Database Setup

## Folder Structure

```

Obsidian/
├── Home.md                 # Master TOC
├── MOCs/                   # Index notes per topic
│   ├── Python.md
│   ├── Stoicism.md
│   ├── Fitness.md
│   └── AI.md
├── Notes/                  # All content notes (flat)
├── Attachments/            # PDFs, images, etc.
└── Templates/
└── Note Template.md    # All-in-one template

```

## How to Use
1. Open `Home.md` as your landing page
2. Click any MOC to browse a topic
3. Create new notes using `Templates/Note Template.md`
4. Link new notes to relevant MOCs

## Template Fields
- **Title**: Note title
- **Source URL**: Original source link
- **Date created**: Auto-filled with {{date}}
- **Type**: book, video, article, podcast, idea
- **Tags**: Your tagging system
- **Status**: Finished / Unfinished
